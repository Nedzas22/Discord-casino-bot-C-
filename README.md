# 🎰 Discord Casino Bot | DSharp+ (C#)

A feature-packed Discord casino bot built with **DSharp+ (C#)**. Play various gambling games, earn balance, level up, and purchase boosters to enhance your gameplay!

## ✨ Features
- **Game Modes**: Crash, Blackjack, Slots, Guess the Number, Roulette, Dice 🎲
- **JSON Storage**: Stores player data (balance, levels, game history)
- **Leveling System**: Players can **level up** to unlock benefits
- **Boosters & Rewards**: Buy boosters to increase winnings
- **Virtual Economy**: Earn, bet, and manage your balance
